# Evaluation Summary

Evaluate performance of a sensitive text classifier and identify bottlenecks.

## Metrics
- Precision: 0.84  
- Recall: 0.89

## Key Finding
Moderate inter-annotator agreement (Kappa ~0.4–0.5) indicates labeling inconsistency.

## Interpretation
The model is performing reasonably well, but its ceiling is constrained by data quality rather than model capability.
